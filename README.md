# CG_3DModel

Keyboard Shortcuts:

Translate:
F1: transl_x (+);
F2: transl_x (-);
F3: transl_y (+);
F4: transl_y (-);
F5: transl_z (+);
F6: transl_z (-);

Arrows - Rotate: 
KEY_RIGHT: rotate_y (+);
KEY_LEFT: rotate_y (-);
KEY_UP: rotate_x (+);
KEY_DOWN: rotate_x (-);

F7: scale_x, scale_y, scale_z (+);
F8: scale_x, scale_y, scale_z (-);
F9: scale_x (+);
F10: scale_x (-);
